# Kinomore
 Full-stack приложение для поиска и просмотра фильмов. Демо: [https://kinomore.vercel.app/](https://kinomore.vercel.app/)
 
# Стек технологий:
### Front-end
  * React + React Hooks
  * TypeScript
  * Redux Toolkit, RTK Query
  * Sass, CSS modules
  * Jest
  * NextJS
  * PWA
  * React-hook-form, Yup
  * Testing Library
  * Storybook
  
### Back-end
  * Express
  * MongoDB
  * JWT
